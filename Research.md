# Research — рынок аналогов для HIVE

**Продукт:** автономная нейрокоманда для лидгена в мессенджерах (Telegram → WhatsApp/MAX/Avito) + организация продаж: парсинг → AI-досье → персональный аутрич → ИИ-агент ведёт диалог → встреча (Zoom/Calendar) → CRM/дожим.

> ⚠️ **Цены — ориентировочные, на начало 2026.** Перед использованием в продажах/КП проверить на сайтах сервисов: тарифы РФ-сервисов меняются часто, зарубежные — в валюте.

---

## Аналог 1 — PhantomBuster 🟡
**Суть:** облачная платформа автоматизаций («фантомов») для скрапинга и действий в соцсетях (LinkedIn, Instagram, Twitter).

**Шаги пользователя:**
1. Выбираешь готовый «фантом» (например, скрап участников группы)
2. Подключаешь аккаунт через session-cookie
3. Задаёшь вход (URL поиска / список)
4. Запускаешь по расписанию в облаке
5. Выгружаешь CSV / отправляешь в CRM по вебхуку

**Монетизация:** SaaS-подписка. ~$56 / $128 / $352 в мес (по часам выполнения и слотам).

**Как адаптировать:** взять идею «облако автоматизаций по расписанию», но заточить под Telegram MTProto + добавить слой AI-досье и диалога, которого у PhantomBuster нет.

---

## Аналог 2 — Apollo.io 🔴
**Суть:** B2B-база контактов (270М+) + sequences для холодного аутрича + AI-ассистент письма.

**Шаги пользователя:**
1. Фильтруешь базу (отрасль, должность, гео)
2. Сохраняешь список, обогащаешь данными
3. Создаёшь sequence (email + LinkedIn касания)
4. AI пишет/персонализирует сообщения
5. Трекаешь открытия/ответы, синк с CRM

**Монетизация:** freemium. Basic ~$49, Professional ~$79–99 /польз./мес.

**Как адаптировать:** их сила — связка «база + досье + sequence + AI-текст». У тебя база живая (из чатов, с болями и цитатами) — это сильнее статичной B2B-базы. Перенести логику sequence (касания/дожим) в Telegram.

## Аналог 3 — SaleBot.pro 🟡
**Суть:** RU-конструктор чат-ботов и автоворонок для Telegram/VK/WhatsApp/Instagram.

**Шаги пользователя:**
1. Собираешь бота в визуальном редакторе (блоки/сценарии)
2. Подключаешь канал (Telegram-бот, VK и т.д.)
3. Настраиваешь автоворонку и рассылки
4. Сегментируешь подписчиков
5. Смотришь аналитику воронки

**Монетизация:** подписка по числу подписчиков/каналов — ~от 800₽ до 5 000₽+ /мес.

**Как адаптировать:** SaleBot работает только через **Bot API** (отвечает тем, кто сам написал боту). Твоё отличие — **userbot, который пишет ПЕРВЫМ** холодным. Это принципиально другой и более ценный сценарий.

## Аналог 4 — BotHelp 🟢
**Суть:** RU-сервис рассылок и автоворонок в мессенджерах (Telegram, WhatsApp, Instagram).

**Шаги пользователя:**
1. Подключаешь каналы
2. Собираешь воронку из блоков
3. Запускаешь массовую рассылку по своей базе подписчиков
4. Сегменты и теги
5. Отчёты по доставке/конверсии

**Монетизация:** подписка по числу подписчиков — ~от 1 990₽ /мес.

**Как адаптировать:** простой UX воронок — хороший ориентир для интерфейса. Но снова: только своя база подписчиков, нет холодного аутрича и нет AI-агента, ведущего диалог.

## Аналог 5 — «Нейроагент» (твой прямой конкурент) 🔴
**Суть:** кастомный AI-агент для Telegram, которого делают под клиента на заказ.

**Шаги пользователя:**
1. Заказываешь разработку агента
2. Тебе настраивают сценарии под продукт
3. Агент ведёт диалоги в Telegram
4. Квалифицирует, передаёт горячих
5. Дорабатывают инструменты по одному

**Монетизация:** ~70 000₽/мес за обслуживание; новые инструменты «по 1 в неделю».

**Как адаптировать:** твоё позиционирование уже построено против него — «78 инструментов сразу, агент включён бесплатно, + конвейер сбора и досье, которых у него нет». Держать это как главный конкурентный нарратив.

## Аналог 6 — TG-парсер + инвайтер (серый сегмент) 🟢
**Суть:** десктоп/веб-софт для парсинга участников чатов, инвайтинга и рассылок (TgParser, Telegram Inviter и аналоги).

**Шаги пользователя:**
1. Указываешь целевые чаты
2. Парсишь участников (имя, username)
3. Фильтруешь (актив/онлайн)
4. Инвайтишь в свою группу / шлёшь рассылку
5. Ротация аккаунтов

**Монетизация:** дёшево — ~от 500 до 3 000₽/мес или разовая лицензия.

**Как адаптировать:** это «дно рынка» — даёт 2 поля и быстрый бан, без AI и досье. Твой апгрейд: 12+ полей, AI-анализ, безопасные лимиты, диалог. Можно делать дешёвый «парсер» точкой входа в воронку (идея №9 из списка продуктов).

## Аналог 7 — Wati / Smartsender (WhatsApp-боты) 🟡
**Суть:** омниканальные чат-боты и рассылки поверх WhatsApp Business API.

**Шаги пользователя:**
1. Подключаешь WhatsApp Business API
2. Собираешь бота/сценарии
3. Рассылаешь по шаблонам (template messages)
4. Операторская панель + CRM
5. Аналитика диалогов

**Монетизация:** ~от $49/мес + плата за сообщения по тарифам WhatsApp API.

**Как адаптировать:** показывает, как легально работать с WhatsApp (только через Business API и шаблоны, нельзя писать холодным первым). Учесть для фазы 2 (WA) — холодный аутрич там невозможен, только тёплая база.

## Вывод — где «дыра» рынка под HIVE
Никто не закрывает **всю цепочку сразу**: холодный аутрич в Telegram (userbot) **+ AI-досье с болями/цитатами + ИИ-агент, реально ведущий диалог до встречи + организация продаж (Zoom/Calendar/CRM/дожим)**.

- Конструкторы ботов (SaleBot, BotHelp, Wati) — только входящие/своя база, без холода и без досье.
- Парсеры — холод есть, но тупой спам без AI.
- Apollo — близко по логике, но это email/LinkedIn и Запад, не RU-мессенджеры.
- Нейроагент — агент есть, но дорого, медленно и без конвейера сбора.

**Позиционирование HIVE:** «единственная автонейрокоманда полного цикла для русскоязычных мессенджеров — от поиска до сделки, с AI-досье и живым диалогом».
