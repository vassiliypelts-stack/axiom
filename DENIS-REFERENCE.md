# Эталон «платформа Дениса» — что повторяем (расшифровка видео-демо, прислал Василий 2026-07-12)

Денис показывал свою платформу лидогена в Telegram. Ниже — разбор его системы на модули + карта «что у AXIOM уже есть / чего не хватает». Сырая расшифровка — в конце файла.

## Модули системы Дениса

### 1. Конвейер сбора клиентов (авто)
- Система **сама находит чаты** по всей нише (по ссылке/ключам → авто-поиск релевантных).
- Авто-вступает, **собирает всех участников**.
- Парсит людей по нужным «ядрам», **сегментирует по сферам** (IT, мероприятия, бизнес, коучинг, финансы; можно ипотека/модульные дома/крипта/кредиты — любая ниша).
- На каждого делает **досье**.
- Масштаб (его пример): 2000 досье → 1200 целевых → 40 чатов в работе → 800 визиток. Из 50 чатов нашёл 24 релевантных.
- **AXIOM:** есть tg_parser (участники/активные/админы), tg_invites, вступление. ЧЕГО НЕТ: авто-поиск чатов по нише/ссылке, сегментация по сферам, генерация досье.

### 2. Досье-карточка (главное)
- Не «юзернейм + номер», а: **аватар, имя, город, сферы, теги, всё что человек писал в чатах, тема, интересы** — сегментировано.
- **Скоринг от AI**, «крючки» (зацепки), **рекомендация подхода** («начать с вопроса о его опыте в маркетинге / попросить совет по трафику»).
- Забирает: что писал + оставил ли **визитку** о себе (на нетворкингах пишут инфо) + **смотрит по другим соцсетям** → AI-анализ персоны, полная картина. «Знаешь о нём больше, чем он сам помнит».
- Модель: **Opus 4.7** для анализа (Денис: «клод-код = терминал, мозги — Sonnet/Opus/любые; я выбираю модель и плачу за токены»; дёшево — дипсик/Haiku, дорого/качество — Opus).
- **AXIOM:** есть CRM «Базы» (36 811 контактов: score/сегмент/пол/район/теги). ЧЕГО НЕТ: анализ «что писал» → боли/желания/крючки/рекомендация, соцсети, источник-чат+ссылка. Это [[dossier-module]] (P0).

### 3. Мониторинг чатов (реалтайм)
- 600 аккаунтов в 600 чатов **слушают**; **триггерные слова** (до 5000) → ловят «хочу купить квартиру по ипотеке, кредит не дают, подскажите банк».
- Агент пишет **с позиции заботы**: «обратись к Василию, у него есть решение, жена так же вопрос решила».
- Ключевое: **скорость + авто-поиск нуждающихся здесь и сейчас**.
- **AXIOM: УЖЕ ЕСТЬ** — tg_listener (слушание по ключам), реалтайм, авто-ответ ИИ (галка на пульте). Довести до уровня Дениса: больше триггеров, тон «заботы».

### 4. Нейроагент с инструментами (78 тулов)
- Денис критикует NeuroAgents: 4 тула за 70к, дорогое тестирование (7000₽/1500 сообщений).
- У него: **11 тулов по коммуникации** (фото-товары, видео-превью, документ, стикер, геолокация, визитки контактов, имитация печати, задержка ответов), **CRM** (сменить этап воронки, авто-теги, скоринг лида, назначить менеджера), **платежи** (подписка, TG Stars, крипта), **встречи/календарь/слоты/эскалация/база знаний/карточка продукта**.
- Агент «Виктория» упакован под женщину 35-40; промпт-инструкция 33-40к символов, **5 траекторий** продаж; сам решает что продавать (Workshop/вайбкодинг/Zoom-запись).
- Агент ведёт диалог, квалифицирует, передаёт, **дожимает**, сам себя ревьюит.
- Выбор модели под задачу, оплата за токены (N8N/Make интеграции).
- **AXIOM:** есть агент, CRM, календарь частично. Это апдейт до OPUS ([[opus-parity-backlog]]).

### 5. Парсинг «откуда угодно» (7+ связок)
- Обычные решения берут только открытые группы с видимыми участниками.
- Денис: сканирует **био → достаёт ссылки** (в т.ч. спрятанные на закрытую группу), находит 1000 похожих ссылок, засылает **агента-премиум** (не похож на бота), мониторит, забирает участников + альтернативные ссылки из био. Авто-поиск релевантных чатов по ссылке.
- **AXIOM:** частично (парсер/инвайты). ЧЕГО НЕТ: bio-скан ссылок, авто-размножение по похожим чатам, агент-разведчик в закрытые.

### 6. Антибан
- Premium-аккаунт 10-15$, нужны десятки; спалил ферму — «сожгут всю»; Opus запрещено кодить спам (конфиденциальность) → «танцы с бубном»; платформа фиксирует каждую ошибку и учится.
- **AXIOM:** antiban Tier 1, прогрев, упаковка — в работе ([[antiban-system]], [[protected-accounts-and-proxy-gotcha]]).

## Бизнес-модель Дениса (для контекста)
200 карточек-досье = 30 000₽ (без рассылки). Пакет Василию+другу: 100к = 10 000 досье (5к+5к) + 2 агента по 70к + любые тулы. Разведка: 200 досье / 200 сообщений / 1 ниша / сегментация 7 дней. Цель конверсии: 15-20% (не 3%) за счёт персонализации по болям/желаниям.

## Вывод: у AXIOM уже есть скелет
Слушатель чатов (✅ есть), CRM/база 36к (✅), парсер (частично), антибан/прогрев/упаковка (в работе), агент (частично). **Главный разрыв до Дениса — ГЛУБИНА досье** (психо-анализ «что писал» → боли/желания/крючки/рекомендация + соцсети) и **авто-обнаружение чатов + сегментация**. С этого и строим [[dossier-module]].
